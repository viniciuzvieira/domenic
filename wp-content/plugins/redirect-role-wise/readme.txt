Redirect the users from one page according to role to different page. 

== Description ==

In this plugin you can set redirection rules from dashboard for all roles of wordpress. You just need to open plugins setting page and select source page name and destination page name for selected role.

For example if admin comes on a paticular page say redirecting page, he will be redirected to page1, if subscriber comes to that page he will redirected to page2.

Note the below points:

1. If multiple roles are assigned to particular user then - the first role will be considered. (ex. if a user have administrator and contributor role then the redirection rule will be applied according to the administrator role)
2. Currently this plugin redirect only pages.

Steps to set rules:

1. Open menu "Redirect Users" from dashboard
2. Click on Add New Redirection.
3. Select source of the page, select roles that should be redirected, select destination page where the roles will be redirected
5. Save the settings

== Installation ==

= Manual Installation =

1. Upload the plugin files to the `/wp-content/plugins/` directory, 
2. Unzip the file
3. Activate the plugin through the 'Plugins' screen in WordPress
